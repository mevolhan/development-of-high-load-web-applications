# Readme
Разработка высоконагруженных веб-приложений 2-ой курс 2-ой семестр
## Лабораторная работа №1: Балансировка и отказоустойчивость с Nginx

### Описание
Реализовано HTTP-приложение на Java Spring Boot с endpoint `/api/counter`, который возвращает JSON с увеличивающимся счётчиком.

### Алгоритмы балансировки
Round Robin (по умолчанию)
Least Connections (least_conn)
IP Hash (ip_hash)
Random (random)

### Отказоустойчивость
Настроены параметры `max_fails=2` и `fail_timeout=30s` для автоматического исключения отказавшего сервера и его восстановления.

### Тестирование
Нагрузочное тестирование проведено с помощью Apache JMeter.

### Структура
`pom.xml` – конфигурация Maven
`Dockerfile` – сборка образа приложения
`docker-compose.yml` – оркестрация контейнеров
`nginx_*.conf` – конфигурационные файлы Nginx для разных алгоритмов
`nginx.conf` – текущий активный конфиг
`results/` – скриншоты и отчёты JMeter
