# Лабораторные работы по высоконагруженным системам

## Лабораторная работа №1: Балансировка и отказоустойчивость с Nginx

**Описание:**  
Разработано Java-приложение на Spring Boot с эндпоинтом `/api/counter`, возвращающим увеличивающийся счётчик. Три экземпляра приложения запущены в Docker, Nginx в роли балансировщика нагрузки.

**Технологии:**  
Java 17, Spring Boot, Docker, Nginx, JMeter

### Алгоритмы балансировки
- Round Robin (по умолчанию)
- Least Connections (`least_conn`)
- IP Hash (`ip_hash`)
- Random (`random`)

### Отказоустойчивость
Настроены параметры `max_fails=2` и `fail_timeout=30s` для автоматического исключения отказавшего сервера и его восстановления.

### Тестирование
Нагрузочное тестирование проведено с помощью Apache JMeter.
